## Task 2 — Login + Forgot-Password Submit Chrome (W-7)

Goal: align both public-page submit affordances with UI-SPEC §7.4 — native gecko-btn + inline gecko-spinner + inline gecko-alert-error. No backend wiring; reused the existing `useState` hooks for loading/error.
